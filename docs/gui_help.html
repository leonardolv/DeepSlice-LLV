DeepSlice GUI Help

Use this page with F1 context help in the app.

Global Ingestion Configuration Prediction Curation Export

Global Shortcuts

  • Ctrl+1..Ctrl+5: jump to workflow steps
  • Ctrl+N: start a new session
  • Ctrl+S: save session
  • Ctrl+O: load session or QuickNII file
  • Ctrl+/: open shortcut cheat sheet
  • F1: open this help page for the current step

Step 1: Ingestion

  • Add files/folders or drag and drop.
  • Use naming helper for _sXXX index parsing.
  • Use sorting/filter to inspect large sets quickly.
  • Check the index table for duplicates and parse warnings.

Step 2: Configuration

  • Select species and review atlas preview card.
  • Set thickness manually or auto-estimate.
  • Use Validate Configuration before prediction.
  • Read ensemble help for speed vs stability tradeoff.

Step 3: Prediction

  • Run Alignment to start inference.
  • Use Cancel Alignment to stop at a safe batch boundary.
  • Monitor phase progress, elapsed time, and ETA.
  • Use console copy/clear for troubleshooting.

Step 4: Curation

  • Review confidence colors and linearity plot.
  • Use context menu or space key to toggle bad flags.
  • Apply outlier detection and angle/spacing tools.
  • Use Undo/Redo and confidence filtering for targeted edits.

Step 5: Export

  • Choose output format and base filename.
  • Export writes primary format plus CSV sidecar.
  • Open folder / copy path for downstream workflows.
  • Generate or preview PDF report with selectable sections.
