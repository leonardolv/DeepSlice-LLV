BioSlide v1.1.0 — Tutorial

A hands-on, step-by-step guide to mastering BioSlide. This updated tutorial now includes detailed, practical use of every major tool, including advanced TBI reviewer export workflows, progress/cancel behavior, and full module-level drilldowns.

15 Lessons Beginner → Advanced ~75 min
Beginner

Lesson 0 — Prerequisites

Before starting, ensure you have:

  • Python 3.10 or later installed
  • At least 8 GB RAM (16 GB+ recommended)
  • Windows 10/11 (64-bit)
  • One or more whole-slide image files (.ndpi, .svs, .mrxs, or .tif)
  • A text editor or IDE (optional, for configuration)
No GPU required. BioSlide works fully on CPU. CUDA GPU acceleration is optional and provides speed improvements for large slides.
Beginner

Lesson 1 — Installation

Set up BioSlide in a clean virtual environment.

1

Clone the Repository

git clone https://github.com/your-org/BioSlide.git
cd BioSlide
2

Create a Virtual Environment

python -m venv .venv
.venv\Scripts\activate
3

Install Dependencies

pip install -r requirements.txt

This installs PySide6, OpenCV, OpenSlide bindings, PyVista, Pydantic, and all supporting libraries.

4

Install OpenSlide (for proprietary WSI formats)

pip install openslide-bin

Required for .ndpi, .svs, and .mrxs files. Standard TIFF files work without it.

5

Launch BioSlide

python main.py

You should see a branded splash screen, then the Launcher Dashboard.

✓ Outcome

BioSlide is installed and the Launcher Dashboard is visible.

Lesson 3 — Create a Project

Projects organize your slides, masks, measurements, and settings in a single .hssp file.

1

New Project

Click New Project on the Launcher Dashboard, or press Ctrl+N.

2

Name & Location

Enter a descriptive project name (e.g., Brain_Study_2024). Choose a save folder.

3

Confirm

Click Create. The Main Window opens with your empty project loaded.

Autosave: BioSlide saves your project every 5 minutes automatically. After a crash, you’ll be offered the option to restore your last autosaved state.

✓ Outcome

A new .hssp project file is created and the Main Window is ready for slides.

Lesson 6 — Segment Tissue

Segmentation identifies tissue regions and separates them from the slide background.

Option A — Automatic Segmentation

1

Switch to the Segmentation Tab

Click the Segmentation tab in the tab bar.

2

Click “Auto Segment”

BioSlide runs the automatic pipeline in the background:

Downsample
→
Dual-Space Detection
→
Morphology Cleanup
→
Contouring

Progress is shown in the status bar. The UI remains responsive during processing.

3

Review Results

Detected slices appear as color-coded overlays on the slide and in the Slice List panel on the right. Each slice gets an ID and thumbnail.

Option B — Manual Tools

If automatic detection misses tissue or includes artifacts, use manual tools:

ToolShortcutWhen to Use
Magic WandWClick to flood-fill a region with consistent staining. Adjust tolerance (0–255) in the Property Panel.
LassoLDraw a freehand polygon around tissue that auto-detect missed.
EraserERemove parts of a mask: brush, path, or polygon modes.

Merge & Split

  • Merge (Ctrl+M): Select 2+ slices → combine into one
  • Split (Ctrl+Shift+S): Separate disconnected regions into individual slices
Caution: Very low minimum area thresholds may produce many small noise artifacts. Start with the default and lower it only if tissue is being missed.

✓ Outcome

All tissue regions are segmented with clean masks. Each slice has an ID, thumbnail, and color overlay.

Intermediate

Lesson 7 — Refine Segmentation

After automatic segmentation, boundaries may need fine-tuning. BioSlide provides interactive GrabCut-based refinement.

1

Open Refinement

Right-click a slice in the Slice List and select Refine Segmentation. The refinement dialog opens.

2

Mark Areas to Keep

Click Mark Areas to Keep to select the green brush. Paint strokes over tissue regions that should be included but were missed.

3

Mark Areas to Remove

Click Mark Areas to Remove to select the red brush. Paint strokes over background regions that were incorrectly included.

4

Watch GrabCut Refine

The mask updates in real-time as GrabCut propagates your corrections to similar pixels. Repeat keep/remove strokes as needed.

5

Apply or Cancel

Click Apply to accept the refined mask, or Cancel to discard changes.

Tip: Start with broad strokes on obvious errors. GrabCut is smart about propagating corrections to neighboring pixels with similar colors.

✓ Outcome

Your segmentation masks have clean, accurate boundaries ready for export or measurement.

Intermediate

Lesson 7A — TBI Reviewer Workflow

The TBI Reviewer is a complete case-review pipeline. This lesson covers the full toolchain: segmentation controls, ROI quality checks, approval flow, export settings, progress/cancel handling, and post-export viewer QA.

Part A — Load and Prepare the Case

1

Open TBI Reviewer

Use the toolbar TBI Reviewer button or press Ctrl+Shift+T.

2

Load a Case Folder and Reviewer Context

Click Browse, select your case folder, then click Load Case. The gallery fills with slice files.

3

Inspect the Warning List First

Use warning items to jump directly to problematic slides. Clicking a warning can open and focus the relevant ROI region automatically.

4

Open a Representative Slice on Canvas

In Gallery View, click the Open: filename button for the slice you want to segment.

Part B — Segment and Validate ROIs

5

Segment in Basic Mode

Adjust Threshold and Merge Radius, then click 1) Segment Current Slice.

6

Validate with ROI List and Canvas Selection

Open the collapsible ROI List (sorted smallest first) to catch tiny outliers quickly. Use S for Selection Tool and confirm list/canvas selection stay synchronized.

7

Apply Across Files

Click 2) Apply Current ROI Layout to All Slices to propagate to the rest of the case. Use Undo Last Apply (Ctrl+Z) if needed.

8

Use Advanced Viewer Tools Only When Needed

Click Show Advanced Controls for measurement (M), midline (L), LUT, scoring, and notes. Keep this collapsed for maximum canvas space during routine segmentation.

Part C — Approval, Export, and Review

9

Approve Slides

Use Approve Current WSI Slide as you progress. The reviewer can auto-advance to the next reviewable slide to accelerate case completion.

10

Set Export Profile and Format

Click Change Export Settings... and choose profile (Fast, Balanced, High) and output format (TIFF or PNG). TIFF is the recommended default for speed and compatibility in this workflow.

11

Run Export and Monitor Progress

Click Export Approved Tissue Slices. Watch synchronized inline + modal progress. ETA shows both remaining duration and expected finish time. Use Cancel Current Process if parameters need correction.

12

Perform QA in Export Review Viewer

Open exported slices in the review viewer (TIFF/TIF/PNG supported). Navigate with arrows, Home/End, thumbnails, and Ctrl+Wheel zoom. Later, use Reopen Last Exported Set to return without rerunning export.

TBI Reviewer Tool Map

GroupToolsWhy It Matters
Core SegmentationThreshold, Merge Radius, Segment Current, Apply Layout, Undo ApplyFast case-wide ROI standardization
ValidationROI List, Selection Tool, warning jump-to-slide, zoom controlsCatches outliers and placement mistakes early
ReviewApprove Current, Approve All, score/notes, sync outputBuilds auditable review results
Export + QAChange Export Settings, Export Approved, progress cancel/ETA, Reopen Last Exported SetPredictable, recoverable export pipeline

✓ Outcome

You can run a complete TBI case workflow: segment, validate, approve, export in one selected format, and review outputs with full traceability.

Lesson 9 — Export Results

Standard Export

1

Open the Export Dialog

Press Ctrl+E or go to Export → Export Slices.

2

Choose a Format

FormatBest For
PNGAlpha transparency, metadata embedding, presentations
JPEGQuick sharing, web (smaller files)
TIFFArchival, lossless, further processing
PDFPublications, printing
3

Select an Export Profile

Built-in profiles: PowerPoint PNG, High Quality TIFF, Web JPEG. Or create a custom profile via the Profile Manager.

4

Set Output Folder

Browse for the destination directory. Configure filename templates with placeholders: {slide_name}, {slice_id}, {index}, {timestamp}.

5

Export

Click Export. Each slice is saved with optional JSON sidecar metadata (source slide, coordinates, algorithm, timestamp).

TBI Reviewer Export (Detailed)

6

Use Single-Format Reviewer Export

In TBI Reviewer, exports are generated in one format per run (TIFF or PNG) to save time. Choose format via Change Export Settings... before exporting approved slices.

7

Prefer TIFF Unless Compatibility Requires PNG

TIFF is typically faster and better for large pathology workflows. Use PNG only when downstream tooling requires it.

8

Verify with Export Review Viewer

After export, open the exported set and verify ordering, metadata, and image quality before moving to animation, collage, or external analysis.

Special Export Options

Feathered Alpha

Enable for PNG to get Gaussian-blurred alpha edges — natural-looking tissue boundaries

Quick Export

Rapid single-image export for fast iteration — right-click a slice

Panel Export

Split a WSI region into printable panels with configurable rows, columns, paper size, and DPI

✓ Outcome

Segmented tissue images are saved with reproducible settings and metadata, and reviewer exports can be QA-checked immediately.

Lesson 11 — Animation Studio

Create cinematic camera animations from whole-slide images for presentations and publications.

Part A — Set Up the Source

1

Switch to the Animation Tab

Click the Animation tab.

2

Load a Source

Load a WSI slide or regular image as your animation canvas.

Part B — Define Regions of Interest

3

Draw ROIs

Use the ROI drawing tools to define areas of interest. Five shape types are available:

ShapeUse Case
RectangleStandard region selection
EllipseOrganic / circular features
PolygonComplex boundaries
FreehandFree-drawn irregular regions
PointSpecific coordinate for zoom target

ROIs can be moved, resized (8 handles), and styled with presets via the ROI Manager.

Part C — Create Camera Keyframes

4

Add Keyframes

Navigate to a view (zoom/pan) and add a keyframe. Each keyframe captures:

  • Center coordinates (x, y)
  • Zoom level
  • Transition duration (seconds)
  • Easing curve (Linear, Ease In, Ease Out, Ease In-Out, Slow, Fast, Bezier)
5

Arrange on the Timeline

Drag keyframes on the Keyframe Timeline to reorder. Click any keyframe to edit its parameters. Use Animation Presets for common patterns (overview scan, ROI zoom).

Part D — Add Branding & Overlays

6

Configure Branding

  • Scale bar — Position, color, thickness, font size; calibrated from MPP
  • Logo watermark — Anchor corner, opacity, max size
  • Text overlays — Title, subtitle, caption with font settings

Save overlay configurations as templates for reuse across projects.

7

Emphasis Effects (optional)

Apply a Spotlight effect to highlight specific ROIs with dimmed surroundings. Additional effects (Vignette, Arrow, Callout, Pulse) are planned for future releases.

Part E — Comparison Modes (optional)

8

Compare Views

Use comparison layouts for side-by-side analysis:

  • Side-by-Side — Two views next to each other
  • Swipe — Draggable divider between two views
  • 2×2 Grid — Four simultaneous views

Part F — Export the Animation

9

Choose Format & Export

FormatUse Case
MP4Standard video for presentations
GIFAnimated loops for web/social
MOVHigh-quality QuickTime video
PNG SequenceIndividual frames for post-processing

Set resolution preset (1080p, 4K, Square) and frame rate.

✓ Outcome

A professional camera animation video showcasing your slide with smooth transitions, branding, and optional effects.

Lesson 12 — 3D Reconstruction

Build a pseudo-3D volume from serial histological sections.

Prerequisites

You need exported PNG slice images from a serial-sectioning experiment (sequential numbered files from the same specimen).

1

Switch to the 3D Reconstruction Tab

Click the 3D Recon tab.

2

Load Slice Images

Select a directory containing your PNG slice files. BioSlide auto-detects slice ordering from filenames using 7 regex patterns (e.g., slice_001.png, brain_s12.png).

3

Sort & Reorder

Verify the slice ordering. Reverse if needed. Check for gaps or duplicates (auto-detected).

4

Preprocess

Configure preprocessing options: grayscale conversion, background removal, intensity normalization, and optional downsampling for faster preview.

5

Align Slices

Choose a feature detector (ORB, SIFT, or AKAZE) and run rigid registration. BioSlide aligns adjacent slices using feature matching with Lowe’s ratio test and RANSAC outlier rejection.

6

Render in 3D

Aligned slices are displayed as textured planes in a PyVista/VTK viewport. Adjust:

  • Slice spacing — Z-distance between planes
  • Opacity — Per-slice or global transparency
  • Transform — Yaw, pitch, roll, scale
  • Show every Nth — Display subset for performance
7

Choose a View Preset

One-click camera presets: Coronal, Sagittal, Dorsal, or custom orientations.

8

Animate & Export

Create 3D animations:

  • Grid → 3D Build — Transition from flat grid to 3D stack
  • Fly-through — Camera travels through the Z-axis
  • Orbit — Camera circles around the volume

Export as MP4 with configurable resolution.

✓ Outcome

A 3D rendering of your serial sections with interactive navigation and exportable animation.

Lesson 13 — Batch Processing

Batch Export

1

Open the Batch Queue

Go to Export → Export Slices with multiple slides loaded.

2

Configure & Start

Select an export profile and output directory. Start the batch job. Each slide is processed sequentially.

3

Monitor Progress

The Batch Queue Dialog shows per-job progress bars. Use the controls:

  • Pause — Temporarily halt processing
  • Resume — Continue from where you paused
  • Cancel — Stop the entire batch

Errors for individual items are displayed in the dialog.

Batch Segmentation

Go to Segmentation → Batch Segment All to apply automatic segmentation to all loaded slides at once.

✓ Outcome

Multiple slides are exported or segmented efficiently with full progress tracking and error handling.

Lesson 14 — Complete Tool Deep Dive

This drilldown is designed for users who already know the basic flow and now need confident, detailed use of every major tool.

Tool Drill Plan

ModuleTools to PracticeValidation Goal

Execution Checklist

  • Run one full segmentation cycle with both auto and manual tools on the same slide.
  • In TBI Reviewer, cancel one long-running operation intentionally, then rerun with corrected settings.
  • Export approved TBI slices as TIFF and reopen them in Export Review Viewer.
  • Create one collage with labels and divider styling suited for publication.
  • Build a 5+ keyframe animation including overlays (scale bar, text, logo).
  • Render a 3D stack, test at least two detector modes, and export one camera animation.
  • Export a statistics CSV and verify downstream readability in your analysis tool.
Tip: If your pipeline is throughput-focused, standardize on TIFF exports for reviewer output and reserve PNG for presentation-specific outputs.

✓ Outcome

You can use the entire BioSlide toolset with operational confidence, not just the beginner path.

Best Practices

  • Save frequently — Ctrl+S saves your project. Autosave runs every 5 minutes as a safety net.
  • Start coarse, then refine — Run auto-segmentation first, then fix issues with manual tools and GrabCut refinement.
  • Use export profiles — Create profiles for your common output needs to avoid reconfiguring every time.
  • Check the Log Panel — Ctrl+L reveals detailed processing messages useful for debugging.
  • Use grid view — Preview tab’s grid mode gives you an overview of all imported slides at a glance.
  • Keep your slides on fast storage — SSD storage dramatically improves tile loading speed for large WSIs.
